# ArcSmith Documentation

Bem-vindo à documentação do ArcSmith publicada via GitHub Pages.

Este site foi configurado com Jekyll e tem a pasta `/docs` como fonte de publicação. Depois de publicar, o link ficará disponível em poucos minutos na página de configurações do GitHub Pages.

## O que está aqui

- `docs/_config.yml`: configuração do site
- `docs/index.md`: página inicial
- `docs/ForgeProtocol.md`: conteúdo existente movido para a fonte do site

## Próximos passos

1. Verifique se o GitHub Pages está usando `main` e `/docs`.
2. Aguarde alguns minutos para o GitHub processar a build.
3. Acesse o link gerado na seção de GitHub Pages do repositório.
